# AI OSI Stack IP Boundaries

This repository documents the open-core civic governance layer of the AI OSI Stack. The following matrix summarises what is distributed here versus what remains proprietary and monetisable.

| Domain | Public (Open-Core) | Proprietary (Private Repositories) |
| --- | --- | --- |
| Governance standards & civic documentation | ✅ docs/, legal guidance briefs, governance handbooks, schemas, and versioned manuscripts | ⛔ Policy automation engines, governance workflow implementations inside the Governance Control Tower™ |
| Validators & integrity tooling | ✅ `_open_tools/` validators, schema sanity checks, ledger exemplars, and documentation build scripts | ⛔ AEIP runtime services, ledger execution engines, integrity orchestration pipelines |
| Ledger exemplars & civic receipts | ✅ `ledger/` sample receipts, walkthrough datasets, integrity notices | ⛔ Production ledger engines, federation bridges, commercial compliance dashboards |
| Release history & provenance | ✅ `versions/`, `meta/`, audits, changelog summaries, civic provenance attestations | ⛔ Governance Control Tower™ operational history, AEIP runtime telemetry, commercial deployment archives |
| Communications & press collateral | ✅ `press-kit/`, repository README, civic announcements | ⛔ Revenue models, client documentation, commercial service catalogues |

> **IP Boundary Statement**  
> AEIP runtime, Governance Control Tower, ledger engines, and analytics dashboards are proprietary and excluded from this repository.
